# Harinandan A — Mechanical Engineering Portfolio

A minimalist, high-performance portfolio website for Harinandan A, a B.Tech Mechanical Engineering student at RV College of Engineering, Bangalore.

## Overview

This portfolio showcases Harinandan's expertise at the intersection of classical mechanical design and modern computation. It highlights projects, internships (like Larsen & Toubro), core skills (Piping Stress Analysis, FEA, CAD), and certifications in a clean, modern, and monochrome aesthetic.

## Features

- **Monochrome & Minimalist UI:** Built with a clean blueprint-grid aesthetic inspired by engineering drawings.
- **Fully Responsive:** Adapts seamlessly to desktop, tablet, and mobile viewing experiences.
- **Dynamic Content Modals:** Interactive pop-ups for exploring internship details and course certificates.
- **Scroll Animations:** Smooth reveal animations as the user scrolls through sections.
- **Custom Cursor:** Interactive cursor accent that adapts to clickable elements.

## Tech Stack

The site is built without heavy frameworks to ensure blazing fast performance and fine-grained control:
- **HTML5:** Semantic structuring using modular components.
- **CSS3:** Custom styling utilizing CSS variables, Flexbox, Grid, and micro-animations.
- **Vanilla JavaScript:** DOM manipulation, Intersection Observers for scroll reveals, and dynamic component loading.

## Local Setup

To run this project locally, simply clone the repository and open `index.html` in any modern web browser.

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/hnaportfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hnaportfolio
   ```
3. Open `index.html` in your browser. Alternatively, serve it using a local web server (like VS Code's Live Server extension) to ensure all dynamically loaded components fetch correctly.
